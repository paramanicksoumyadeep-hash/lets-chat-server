
# 💬 Let’s Chat – Server (Backend)

The **Let’s Chat Server** is the backend of a real-time chat application built using **Node.js** and **Socket.IO**.  
It manages WebSocket connections and enables real-time message exchange between connected clients.

🌐 **Server URL:** https://lets-chat-server.onrender.com/

---

## 🔍 Overview

This server handles:
- Client connections and disconnections
- Real-time message broadcasting
- WebSocket event handling using Socket.IO
